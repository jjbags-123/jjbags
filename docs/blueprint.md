# **App Name**: JJ Bages

## Core Features:

- Product Catalog: Display categorized jute, juco, and tote bags with details like size and material. Product data comes from a structured file.
- Dynamic Blog: A blog page to publish articles related to eco-friendly living and sustainable fashion.
- Corporate Order Form: Allow businesses to submit custom orders with specific requirements for bulk purchases.
- Image Gallery: Showcase high-quality images of bags in different settings to inspire customers.
- AI-Powered Product Recommendation: Suggest relevant products to the user based on the content they are currently viewing using an AI tool.
- Contact Form with Inquiry Routing: Enable users to submit inquiries, routed to the appropriate team based on the inquiry type.
- Order Tracking: Allow logged-in users to view their order history and track the status of current orders.

## Style Guidelines:

- Primary color: Earthy green (#8FBC8F), evoking nature and sustainability.
- Background color: Off-white (#F5F5DC), provides a clean and eco-friendly feel.
- Accent color: Natural brown (#D2B48C) for highlighting important elements and CTAs.
- Headline font: 'Belleza' serif, for an elegant style suitable for fashion, art, and design; body font: 'Alegreya' serif, suitable for blocks of text.
- Use minimalist, line-style icons representing nature, recycling, and bag types.
- Employ a clean, grid-based layout with plenty of white space to enhance readability and create a modern feel.
- Subtle animations on page transitions and image hovers to add a touch of sophistication without sacrificing performance.
